# Power Membership | 讓每個人都可以輕鬆建立會員制網站
一句話講完 Power Membership :

> Power Membership 是一個 WordPress 套件，安裝後，可以讓你輕鬆創建會員制網站。

<br><br><br>

# Features

## 會員等級
 - 預設創建 Gamipress Rank Type `member_lv`
 - 在 `會員等級` 頁面，可以設定用戶需要 `累積消費` 到多少金額才可以升級到這個等級


## 會員生日
 [ ] - 後台欄位新增生日
 [ ] - 註冊新增生日欄位
 [ ] - 儲存欄位
